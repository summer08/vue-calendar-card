# vue实现日历卡片
本项目在创建完脚手架的基本上实现一个移动端日历卡片的效果，日历显示本年份中当前月份之前的月份包含当前月

## Build Setup
``` bash
# install dependencies
npm install

# Compiles and hot-reloads for development
npm run serve

# test环境
npm run test

# Compiles and minifies for production
npm run build

# Lints and fixes files
npm run lint
```
